Proyectos Destacados

IT Department Data Analysis

En este proyecto, trabajé con un conjunto de datos que incluye 12 meses de información presupuestaria y de pronóstico, así como 6 meses de datos reales relacionados con el departamento de TI de una empresa global. El objetivo principal fue identificar tendencias, comparar el rendimiento real con el presupuesto y proporcionar información para la toma de decisiones estratégicas.
